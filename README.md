# Sudoku Vibe

A mobile-first Sudoku PWA built with React, Tailwind CSS, and Vite.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy To Vercel

Import this GitHub repository in Vercel. Use the default Vite settings:

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
